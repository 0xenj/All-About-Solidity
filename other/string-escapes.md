| Escape type  	| Description  	|
|:-:	|:-:	|
| `\<newline>`      | escape an actual new line      |
| `\\`      | backslash      |
| `\'`       | single quote      |
| `\"`       | double quote      |
| `\b`      | backspace      |
| `\f`       | form feed      |
| `\n`       | newline      |
| `\r`       | carriage return      |
| `\t`       | tab      |
| `\v`       | vertical tab      |
| `\xNN`       | hex escape      |
| `\uNNNN`       | unicode escape      |
